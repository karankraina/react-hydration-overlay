## Usage

Use the plugin in Webpack. Pass your application query selector in the constructor.

```sh
const { Overlay } = require('react-hydration-overlay');

// ...

export default {
    // ...,
    plugins: [
        // ...,
        new Overlay('div#app'),
    ]
}
```
